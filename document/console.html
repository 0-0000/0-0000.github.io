miniExtend Console

这里的 Console 并不指真实的程序控制台，而是日志，在程序控制台输出信息对开发者而言几乎是无意义的，下文将日志称为控制台。

Console 作用域包含以下函数：

函数 描述
log(...) 在控制台格式化 ... 输出，格式化方式与 lua 标准函数中的 print 相同，标签为 "global" 。
logtag(tag, ...) 在控制台格式化 ... 输出，标签为 tag。
warn(message) 在控制台输出警告信息 message ，标签为 "warning" 。
warn(message) 在控制台输出错误信息 message ，标签为 "error" ，该函数不会抛出错误。
clear() 清空控制台。
