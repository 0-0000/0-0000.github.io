miniExtend Object

目前 Object 中的内容极少，目前的作用是设置 objid 默认参数。
在回调函数（使用 miniExtend 函数绑定）中，通常只操作事件中的对象 (eventobjid) ，少数情况下操作目标对象 (toobjid) ，随机对象和所有类型对象（例如全体玩家，同一队伍玩家）。
而 objid 在前两种情况下作用很大，特别是 eventobjid ，例如在回调函数中调用 CustomUI 中的函数，无需传实参，默认使用 eventobjid。
更多内容参见 miniExtend Event 和 miniExtend CustomUI 。
注意， objid 并非全局变量，它实际上是 _G2["__objid"] ，应该使用 getObjectId() 获取它的值。

函数 描述
getObjectId() 返回 objid 的值。
useObjectId(objid) 设置 objid 的值，如果参数是一个表，则设置为 "eventobjid" 键的值，如果参数是数字，直接设置，否则不做任何有意义的操作。
